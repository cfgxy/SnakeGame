## 🚀 运行方式

### 运行应用层测试
```bash
cd /usr/local/src/SnakeGame
dotnet test tests/SnakeGame.App.Tests/
```

### 运行性能基准测试
```bash
cd /usr/local/src/SnakeGame
dotnet run --project tests/SnakeGame.Benchmarks/
```

## 📊 性能测试指标

### 目标指标
| 指标 | 目标值 | 说明 |
|------|--------|------|
| GameEngine.Step | < 0.1ms | 单帧处理时间 |
| GC 每帧 | < 1ms | 垃圾回收时间 |
| 内存分配 | < 1KB/帧 | 每帧内存分配 |
| FPS | 稳定 60 | 帧率稳定性 |

### 测试场景
1. **GameEngine.Step** - 单帧更新性能
2. **CreateSession** - 会话创建性能
3. **蛇身数据结构** - List vs Array vs ImmutableArray
