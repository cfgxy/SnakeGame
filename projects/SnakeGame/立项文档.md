## 一、项目概述

### 1.1 项目名称
**SnakeGame** - 贪吃蛇游戏

### 1.2 项目定位
一个基于 **MonoGame + C# + .NET 8** 的中文贪吃蛇实验项目，旨在演示 **VibeCoding + TDD（测试驱动开发）** 下的开发流程。

### 1.3 核心目标
- 验证 AI 协作编码流程
- 演示 TDD 驱动核心规则落地
- 建立从需求、计划、测试到实现的完整可追踪链路

### 1.4 当前状态
- 🆕 新建项目（2026-03-20 创建）
- 已完成 V1.0 基础版本
- 代码已开源，可公开访问

## 三、项目结构

```
SnakeGame/
├─ docs/                    # 文档目录
├─ src/
│  ├─ SnakeGame.Core/     # 核心业务逻辑层
│  └─ SnakeGame.App/      # 应用层（入口）
├─ tests/
│  └─ SnakeGame.Core.Tests/  # 核心层单元测试
└─ .github/
   └─ workflows/          # GitHub Actions 工作流
```

### 3.1 核心模块说明

| 模块 | 说明 |
|------|------|
| **SnakeGame.Core** | 核心游戏逻辑、规则、状态管理 |
| **SnakeGame.App** | 游戏入口、渲染、输入处理 |
| **SnakeGame.Core.Tests** | 核心层单元测试 |
