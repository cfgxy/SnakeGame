## 技术亮点

### 1. 自动化资源生成

使用 Python PIL 脚本自动生成所有视觉资源：
- 蛇：圆润造型 + 眼睛
- 食物：珊瑚红 + 高光 + 叶梗
- 背景：天空蓝→樱花粉渐变
- UI：紫罗兰按钮（4 状态）

### 2. 分层渲染架构

```
1. BackgroundLayer → 渐变背景
2. GridLayer → 地面网格
3. FoodLayer → 食物（带动画）
4. SnakeLayer → 蛇（头/身/尾）
5. ParticleLayer → 粒子特效
6. HUDLayer → 分数显示
7. UILayer → 界面
```

### 3. 粒子系统

- 对象池管理
- 生命周期控制
- 缩放渐变效果
- 吃食物/死亡触发
