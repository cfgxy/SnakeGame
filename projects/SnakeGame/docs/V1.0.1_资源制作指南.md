## 四、制作流程

### 4.1 使用 Aseprite 制作

1. **新建文件**: 按上述尺寸创建
2. **设置网格**: View → Show Grid（32px 或 28px）
3. **绘制造型**: 使用铅笔/矩形/圆形工具
4. **添加细节**: 眼睛、高光、叶梗
5. **导出**: File → Export → PNG（透明背景）

### 4.2 使用 Figma 制作

1. **新建 Frame**: 按尺寸创建
2. **绘制形状**: Rectangle/Ellipse 工具
3. **设置圆角**: Corner Radius 8px
4. **填充颜色**: 使用上述色板
5. **导出**: 右上角 Export → PNG

### 4.3 使用 Piskel（在线）

1. 访问 https://www.piskelapp.com/
2. 创建新 Sprite（设置尺寸）
3. 使用像素画工具绘制
4. 下载 PNG

## 六、质量检查

### 检查清单

- [ ] 所有 PNG 使用透明背景

1. 将 PNG 放入 `Content/` 对应目录
2. 运行游戏查看效果
3. 检查透明背景是否正常
4. 验证颜色和尺寸
